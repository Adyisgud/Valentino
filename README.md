A cute interactive Streamlit web app to ask someone to be your Valentine!

## Features
- Responsive design for desktop and mobile
- Interactive "Yes" and "No" buttons
- "No" button moves around when clicked
- Success message with balloons when "Yes" is clicked

## Running Locally

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run app.py
```

## Deploying to Streamlit Cloud

1. Push this code to a GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub
4. Click "New app"
5. Select your repository and branch
6. Set main file path to `app.py`
7. Click "Deploy"!

Your app will be live in a few minutes with a free URL you can share!

## Customization

- Replace the image URLs in `app.py` with your own images
- Modify the colors and styling in the CSS section
- Change the success message text
